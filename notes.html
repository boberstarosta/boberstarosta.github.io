Starting Django project

Install virtualenv and virtualenvwrapper $ pip install virtualenvwrapper-win $ mkvirtualenv project-name $ workon project-name

Install Django $ pip install django $ django-admin --version

Start Django project $ django-admin startproject project-name

Managing Django project

Start Django development server (localhost:8000) $ python manage.py runserver

Create Django app $ python manage.py startapp app-name Run python interactive shell $ python manage.py shell

Database

Create migration $ python manage.py makemigrations polls Migrate $ python manage.py migrate

Django Admin

Create an admin user $ python manage.py createsuperuser
