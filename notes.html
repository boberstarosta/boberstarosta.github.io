Starting Django project

Install virtualenv and virtualenvwrapper
$ pip install virtualenvwrapper-win
$ mkvirtualenv project-name
$ workon project-name

Install Django
$ pip install django
$ django-admin --version

Start Django project
$ django-admin startproject project-name

Managing Django project

Start Django development server (localhost:8000)
$ python manage.py runserver

Create Django app
$ python manage.py startapp app-name Run python interactive shell
$ python manage.py shell

Database

Create migration
$ python manage.py makemigrations polls Migrate
$ python manage.py migrate

Django Admin

Create an admin user
$ python manage.py createsuperuser

Git

Setting up a new repo

Change to project directory
$ cd project-dir Create a new repo
$ git init

Commiting

Add a file to staging
$ git add file-name Show status of current commit
$ git status Unstage a file
$ git rm --cached file-name Commit!
$ git commit -m "commit message"

Branches

Create a new branch
$ git checkout -b branch-name Switch current branch
$ git checkout branch-name Show branch info
$ git branch Delete branch
$ git branch -d branch-name

Remote

Setup github repo
$ git remote add origin github.repo.url Push the repo to github
$ git push -u origin branch-name Pull the repo from github
$ git pull origin branch-name Clone repo from github to current folder
$ git clone github.repo.url
